## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```sh
   git clone <repository-url>
   cd Mix_Cafe_App
   ```

2. **Install dependencies:**
   ```sh
   flutter pub get
   ```

3. **Run the app:**
   ```sh
   flutter run
   ```
